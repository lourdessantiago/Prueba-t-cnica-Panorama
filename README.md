# Prueba técnica PANORAMA
![1.png](https://i.postimg.cc/5N7GLfwM/1.png)
## Resumen del proyecto

Es un proyecto desarrollado por medio de HTML5, CCS y JavaScript. Donde se realiza el consumo de una API en la cual se pueden observar 323 productos con su imagen y nombre, al dar click en algún producto se observan los detalles de cada producto, así como realizar búsquedas y obtener resultados a partir de los 3 primeros caracteres.

## Instrucciones para levantar el proyecto

 - Ir a la dirección del siguiente repositorio [https://github.com/lourdessantiago/Prueba-t-cnica-Panorama](https://github.com/lourdessantiago/Prueba-t-cnica-Panorama)
 - Dar click en clonar el repositorio.
 - Abrir visual studio code.
 - Dar click en “Clonar el repositorio Github”.
 - Pegar el link y crear la carpeta donde se guardará el proyecto.
 - Abrir terminal e instalar dependencias con npm Install.
 - Correr el proyecto con npm start.
 - Abrir en el navegador con la liga que generó.
